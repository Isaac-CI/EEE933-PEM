O repositório EEE933-PEM contém o projeto final do grupo B composto por Guilherme Alvarenga Argueso, Gustavo Duarte Tomaz de Sá, Isaac Nóbrega Marinho e Leticia Vitoria Souza Pires, para a disciplina de Processos e Equipamentos para Microeletrônica.

O projeto final é composto pelos arquivos de descrição da simulação pelo Silvaco do processo de fabricação e das propriedades elétricas de um resistor de 1 K$\Omega$ poly-poly, um capacitor de 1pF poly-poly, um transistor NMOS de 180nm e uma porta inversora CMOS de 180nm.
